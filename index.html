Jmath

JMath is an extended mathematics utils Java library

View project on GitHub

Welcome to JMath, the extended mathematics library for Java!

JMath provide a bunch of mathematics oriented functionnalities (e.g. function derivatives, analytical geometry, matrix calculations etc. (see Wiki if you want the whole list of features)).

Installing

For the moment, the only way to use JMath is to clone and build it by yourself, the time I set up a JAR library and a Maven repository.

Contributing

Everyone can contribute to JMath by adding features, correcting bug; the only rules are:

  • You must push you modifications on a separated Git Branch
  • You must write JUnit tests for all your classes/features
  • Your tests all must pass before pushing onto your branch
